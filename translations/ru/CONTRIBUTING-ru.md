# Контрибььюция

## Pull request'ы
- Если шаблон PR обновляется, все существующие PR обязаны соответствовать обновленному шаблону, чтобы быть объедененными
- Мы оставляем за собой право отклонить любой pull request по любой причине

## Issues
- **Сообщить о абьюзивном домене** - Создайте новый issue с меткой `report-abuse`.
- *Сообщить о проблеме с вашим доменов* - Создайте новый issue с меткой `support`.
- **Предложения** - Создайте новый issue с меткой `suggestion`.
- **По любым вопросам** - Создайте новый issue с меткой `question`.
- **Сообщить о любых других проблемах, связанных с этим репозиторием** - Создайте новый issue с подходящей меткой.
